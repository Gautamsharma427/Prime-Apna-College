# Basic Terminal Commands for Computer Engineers

## Navigation
- `pwd` - Print working directory
- `cd [path]` - Change directory
- `ls` - List files and folders
- `ls -la` - List with hidden files and permissions

## File Operations
- `touch [file]` - Create a new file
- `mkdir [dir]` - Create a directory
- `cp [source] [dest]` - Copy files
- `mv [source] [dest]` - Move or rename files
- `rm [file]` - Remove files
- `rm -r [dir]` - Remove directory recursively

## File Viewing
- `cat [file]` - Display file contents
- `less [file]` - View file page by page
- `head [file]` - Show first 10 lines
- `tail [file]` - Show last 10 lines

## System Information
- `whoami` - Current user
- `date` - Current date and time
- `df -h` - Disk space usage
- `top` - Running processes

## Permissions & Ownership
- `chmod [perms] [file]` - Change permissions
- `chown [user] [file]` - Change owner

## Text Processing
- `grep [pattern] [file]` - Search text
- `sed` - Stream editor for text
- `awk` - Text processing tool

## Useful Shortcuts
- `Tab` - Auto-complete
- `Ctrl+C` - Cancel command
- `Ctrl+L` - Clear screen
- `history` - Command history
- `touch <filename>` - Creates a file with respective extension
